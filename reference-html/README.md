# Reference HTML

Folder ini berisi prototype HTML asli yang dipakai sebagai referensi visual.

Isi folder:
- `dashboard.html`
- `projects.html`
- `identify.html`
- `diagnosis.html`
- `objective.html`
- `campaign.html`
- `settings.html`
- `shared-responsive.css`

Catatan:
- File-file ini masih bersifat statis dan dipakai sebagai acuan layout.
- `index.html` di root akan redirect ke `reference-html/dashboard.html`.
- Kalau nanti kita pindah ke Next.js, folder ini bisa tetap dipakai sebagai referensi desain.
