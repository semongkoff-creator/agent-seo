SEO Agent - Reference Prototype + Next.js Scaffold
===================================================

Prototype HTML sekarang disimpan di folder `reference-html/`.
Root project berisi fondasi Next.js dan helper awal untuk rewrite bertahap.

QUICK START:
  - Buka `index.html` untuk redirect ke `reference-html/dashboard.html`
  - Buka file di `reference-html/` langsung di browser jika ingin melihat prototype
  - Jalankan `pnpm dev` setelah dependencies dipasang

STRUKTUR UTAMA:
  reference-html/              prototype HTML asli
  app/                         fondasi Next.js App Router
  components/                  komponen bersama
  lib/                         helper backend dan utilitas

CATATAN:
- Prototype HTML tetap dipertahankan sebagai visual reference.
- Struktur Next.js masih bertahap dan belum semua route diimplementasikan.
